Docker Commands Quick Reference

Image Management

Command Description Example
docker build -t NAME:TAG . Build an image from Dockerfile in current directory docker build -t k8s-demo:v1 .
docker images List all images docker images | grep k8s-demo
docker rmi IMAGE Remove an image docker rmi k8s-demo:v1
docker tag SOURCE TARGET Create a new tag for an image docker tag k8s-demo:v1 k8s-demo:latest

Container Management

Command Description Example
docker run -d -p HOST:CONTAINER --name NAME IMAGE Run container in background with port mapping docker run -d -p 8080:8080 --name app k8s-demo:v1
docker ps List running containers docker ps
docker ps -a List all containers (including stopped) docker ps -a
docker stop CONTAINER Stop a running container docker stop k8s-demo-container
docker start CONTAINER Start a stopped container docker start k8s-demo-container
docker rm CONTAINER Remove a stopped container docker rm k8s-demo-container
docker rm -f CONTAINER Force remove a running container docker rm -f k8s-demo-container

Debugging & Inspection

Command Description Example
docker logs CONTAINER View container logs docker logs k8s-demo-container
docker logs -f CONTAINER Follow (tail) container logs docker logs -f k8s-demo-container
docker exec -it CONTAINER /bin/sh Execute interactive shell in running container docker exec -it k8s-demo-container /bin/sh
docker inspect CONTAINER View detailed container information (JSON) docker inspect k8s-demo-container

Common Flags

Flag Description
-d Detached mode (run in background)
-p HOST:CONTAINER Port mapping (8080:8080 maps host 8080 to container 8080)
--name NAME Assign a name to the container
-it Interactive terminal (for exec/run)
-f Force (for rm, logs -f for follow)
--rm Automatically remove container when it stops
-e KEY=VALUE Set environment variable

💡 Pro Tips

  • Clean up everything: docker system prune -a (removes unused images, containers, networks)
  • View image layers: docker history IMAGE
  • Copy files from container: docker cp CONTAINER:/path/to/file ./local-path
  • Quick test without name: docker run --rm -p 8080:8080 k8s-demo:v1 (auto-removes on stop)
