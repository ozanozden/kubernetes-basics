# Docker and Kubernetes Resources

## Knowledge

- [Docker Official Documentation - Get Started](https://docs.docker.com/get-started/)
  Official Docker guide covering containers, images, Dockerfile best practices. Use for: Dockerfile syntax, image layering, multi-stage builds.

- [Kubernetes Official Documentation - Tutorials](https://kubernetes.io/docs/tutorials/)
  Canonical K8s learning path. Use for: pod/deployment/service concepts, kubectl commands, architecture.

- [Spring Boot with Docker Guide](https://spring.io/guides/gs/spring-boot-docker)
  Official Spring guide for containerizing Boot apps. Use for: Boot-specific Dockerfile patterns, jar packaging.

- [Minikube Documentation](https://minikube.sigs.k8s.io/docs/start/)
  Local K8s cluster setup and usage. Use for: cluster management, troubleshooting, addons.

## Wisdom (Communities)

- [r/kubernetes](https://reddit.com/r/kubernetes)
  Active community for K8s questions and patterns. Use for: troubleshooting, best practices.

- [Stack Overflow - Docker tag](https://stackoverflow.com/questions/tagged/docker)
  Large corpus of solved Docker issues. Use for: specific error resolution.
